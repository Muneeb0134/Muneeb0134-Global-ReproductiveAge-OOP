# Global-ReproductiveAge-OOP
Headline: 🌍 Global Reproductive Health Analysis: A Data-Driven Journey using Python & OOP

I am excited to share my latest Data Science project where I analyzed global reproductive health trends by merging WHO datasets with Life Expectancy statistics.

As a Computer Science student, my goal was not just to write code, but to uncover the "why" behind healthcare disparities across different regions. Using Object-Oriented Programming (OOP), I built a scalable analyzer to clean, process, and visualize complex data.

Key Insights from the Analysis: ✅ The Healthcare Gap: Regions like Central America and parts of Asia are showing impressive strides in healthcare satisfaction, often outperforming expectations relative to their economic status. ✅ The Correlation: My scatter plot analysis confirms a strong link between Life Expectancy and modern family planning satisfaction—proving that integrated health policies are the backbone of longevity. ✅ Identifying Intervention Areas: Despite global progress, significant gaps remain in several regions, highlighting the need for targeted healthcare reforms.

Tech Stack Used: 🔹 Language: Python 🔹 Libraries: Pandas (Data Wrangling), Seaborn & Matplotlib (Visualizations) 🔹 Architecture: Object-Oriented Programming (OOP) for modular and clean code.
